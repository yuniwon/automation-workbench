# Pricing Guide

목표: Gmail로 실제 문의가 왔을 때 흔들리지 않고 범위와 가격을 안내한다.

이 문서는 공개 가격표가 아니라 내부 견적 기준이다. 첫 수주 전까지는 낮은 마찰,
명확한 범위, 빠른 납품을 우선한다.

## 참고한 시장 신호

- 국내 크몽의 엑셀 매크로/VBA 예시는 기본/중급/고급 패키지를 5만 원, 15만 원, 50만 원으로 나누고, 작업일을 1일/5일/15일로 구분한다.
- 해외 Excel automation 컨설팅 자료는 단순 수정, 단일 리포트, 데이터 정리 포함 리포트, 다중 소스 시스템처럼 복잡도 기준으로 가격이 커진다고 설명한다.
- Guru의 Excel automation consultant 안내는 시간당 25~100달러 범위를 제시하지만, 초기 수주에서는 시간당 과금보다 고정 범위 견적이 설명하기 쉽다.

## 견적 원칙

1. 시간당 단가보다 결과물 기준 고정가를 먼저 제시한다.
2. 샘플 파일을 보기 전에는 확정 견적을 내지 않는다.
3. 첫 수주 전까지는 포트폴리오 확보를 위해 낮은 진입 가격을 허용한다.
4. 작업 범위, 입력 파일, 결과 파일, 수정 횟수, 납품 방식을 문장으로 합의한다.
5. 외부 서비스 연동, 로그인, 결제, 개인정보 처리, 예약 실행은 별도 범위로 본다.

## 첫 수주용 패키지

### Starter: 50,000원부터

조건:

- 입력 파일 1개
- 규칙 3개 이하
- 결과 CSV 또는 XLSX 1개
- 외부 서비스 연동 없음
- 예상 납기 1일

예시:

- 중복 제거
- 빈 값 표시
- 열 이름 정리
- 숫자 형식 정리
- 특정 열 기준 정렬

### Standard: 150,000원부터

조건:

- 입력 파일 2개 이하
- 비교, 병합, 요약 중 하나 포함
- 결과 파일 또는 요약 리포트 1개
- 예외 규칙이 있지만 설명 가능
- 예상 납기 2~5일

예시:

- 주문 파일과 정산 파일 비교
- 여러 매장 파일 병합
- 월간 요약표 생성
- 누락/변경 행 리포트 생성

### Advanced: 300,000원부터

조건:

- 입력 파일 3개 이상
- 복수 단계 처리
- 사용자별 예외 규칙
- 결과물 검수 화면 또는 사용 설명서 필요
- 예상 납기 5~10일

예시:

- 주문, 정산, 재고 파일을 함께 처리
- 고객사 양식별 결과 파일 생성
- 반복 사용 가능한 웹 도구화
- 리포트와 검수표 동시 생성

### Custom: 별도 산정

조건:

- 구글시트, Gmail, Notion, Slack 등 외부 서비스 연동
- 예약 실행 또는 알림 발송
- 로그인/권한/개인정보 처리 포함
- 운영 유지보수 필요

## 첫 답장에 넣을 견적 문장

```text
샘플 파일을 확인한 뒤 확정 견적을 드리겠습니다.

현재 기준으로는 작업 범위에 따라 보통 아래처럼 나뉩니다.

- 단일 파일 정리/변환: 5만 원부터
- 두 파일 비교/병합/요약: 15만 원부터
- 여러 파일과 예외 규칙이 있는 반복 도구: 30만 원부터
- 외부 서비스 연동이나 예약 실행: 별도 산정

정확한 금액은 입력 파일, 결과물 예시, 반복 작업 설명을 보고 안내드리겠습니다.
```

## 견적 전 필수 질문

1. 현재 파일 형식은 무엇인가: CSV, XLSX, 구글시트, 기타
2. 입력 파일은 몇 개인가
3. 매번 반복하는 작업은 무엇인가
4. 결과물은 어떤 형태여야 하는가
5. 현재 수작업은 얼마나 걸리는가
6. 샘플 파일 공유가 가능한가
7. 개인정보나 민감정보가 포함되는가
8. 희망 마감일이 있는가

## 거절 또는 보류 기준

- 샘플 파일 없이 확정 견적을 요구한다.
- 결과물 기준이 계속 바뀐다.
- 개인정보 처리가 큰데 보안 요구사항이 불명확하다.
- 외부 계정 권한을 맡기면서 책임 범위를 합의하지 않는다.
- 납기 대비 범위가 과도하다.
