# Gmail Intake Playbook

목표: 무료 도구 사용자가 `dnjsdndus@gmail.com`으로 문의했을 때 빠르게 자격을 판단하고,
샘플 파일 확인, 견적, 제작 범위 확정까지 이어지게 한다.

## 검색 기준

Gmail에서 우선 확인할 검색어:

```text
("엑셀" OR "CSV" OR "XLSX" OR "자동화" OR "정리" OR "비교" OR "automation-workbench")
```

강한 구매 신호:

- 샘플 파일을 첨부했거나 공유 가능하다고 적음
- 반복 작업 설명이 구체적임
- 결과 파일 양식이나 보고서 예시가 있음
- "견적", "제작", "가능한가요", "자동으로" 같은 표현이 있음

낮은 우선순위:

- 파일 구조 설명이 없음
- 자동화가 아니라 단순 사용법 질문만 있음
- 작업 범위가 너무 넓고 결과물이 불명확함

## 상태 분류

```text
NEW              새 문의
NEEDS_SAMPLE     샘플 파일 또는 화면 캡처 요청 필요
QUALIFIED        파일 구조와 반복 작업이 확인됨
QUOTED           견적과 작업 범위 안내 완료
WON              진행 확정
LOST             보류 또는 무응답
```

## 첫 답장 템플릿

제목:

```text
Re: 엑셀/CSV 자동화 맞춤 제작 문의
```

본문:

```text
안녕하세요.
문의 감사합니다.

정확히 가능 범위와 견적을 보려면 아래 항목만 확인하면 됩니다.

1. 현재 사용 중인 파일 형식: CSV / XLSX / 구글시트 / 기타
2. 반복해서 하는 작업: 예) 중복 제거, 정산 파일 비교, 특정 양식으로 변환
3. 현재 수작업 소요시간: 예) 매주 2시간, 매일 30분, 건당 5분
4. 최종 결과물: 예) 다운로드할 CSV, 엑셀 리포트, 구글시트 자동 업데이트
5. 희망 마감일: 급한 일정이 있으면 먼저 적어주세요.
6. 샘플 파일 공유 가능 여부: 실제 데이터가 어렵다면 일부 값은 가린 샘플도 괜찮습니다.

샘플 파일을 기준으로 작업 범위와 예상 금액을 안내드리겠습니다.
```

## 견적 판단 기준

간단한 작업:

- 한 파일 기준 정리 또는 변환
- 규칙이 3개 이하
- 결과 CSV/XLSX 하나
- 예상 작업: 반나절 이내

중간 작업:

- 두 파일 이상 비교 또는 병합
- 예외 규칙이 있음
- 결과 리포트 또는 요약표 필요
- 예상 작업: 1~2일

복잡한 작업:

- 구글시트, Gmail, Slack, Notion 등 외부 서비스 연동
- 주기적 실행 또는 알림 필요
- 고객별 설정 화면 필요
- 예상 작업: 별도 범위 산정

## 진행 확정 전 확인할 것

- 샘플 파일을 로컬에서 열 수 있는가
- 개인정보나 민감정보가 포함되어 있으면 가림 처리됐는가
- 입력 파일과 결과 파일의 성공 예시가 있는가
- 현재 수작업 소요시간과 반복 주기가 확인됐는가
- 희망 마감일이 작업량과 맞는가
- 작업 완료 기준이 문장으로 합의됐는가
- 납품 방식이 명확한가: 웹 도구, 스크립트, 구글시트 자동화, 사용 설명서 등

```text
짧게 한 문장 후기를 남겨주시면 이후 도구 개선과 포트폴리오에 큰 도움이 됩니다.
```
