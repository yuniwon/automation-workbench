# Upwork Proposal Draft: QR Vendor Check-In

Use this with the QR vendor check-in job or similar Google Sheets + Apps Script
workflow jobs.

## Short Proposal

```text
Hi,

I can build this as a lightweight Google Sheets + Apps Script check-in system for all 4 properties.

For your workflow, I would create one Google Sheet, deploy a mobile-friendly Apps Script Web App, and generate one property-specific check-in link per property. When a vendor scans the QR code, the property would already be selected, and their service type, company name, notes, timestamp, and property would be saved to the sheet. I would also add an email notification for each submission.

I recently built a similar demo here:
https://yuniwon.github.io/automation-workbench/?lang=en&tool=checkin&source=upwork-portfolio

For this job I would deliver:
- Google Sheet with Check-ins, Properties, and Settings tabs
- Apps Script Web App form
- 4 property-specific QR links
- email notification on each check-in
- simple setup and usage instructions

Estimated fixed price: $300
Estimated delivery: 3 days

One quick question: should all check-in notifications go to one email address, or should each property have its own manager email?

Best,
Yuni
```

Best,
