# Upwork Job Reviews

이 폴더는 Upwork에서 발견한 실제 공고를 검토하고, 재사용 가능한 제안서와
포트폴리오 샘플 아이디어로 남기는 곳이다.

운영 원칙:

- 공고 원문과 숫자는 시간이 지나면 바뀔 수 있으므로 snapshot 날짜를 남긴다.
- Connects를 쓰기 전에는 현재 공고 상태를 다시 확인한다.
- 지원 버튼은 사용자가 최종 승인한 뒤 직접 누른다.
- 좋은 공고는 지원하지 않더라도 포트폴리오 샘플 요구사항으로 재사용한다.
