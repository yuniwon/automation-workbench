# Automation Workbench v0.1.0

무료 엑셀/CSV 정리·비교 도구의 첫 공개 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 주요 기능

- CSV/XLSX 파일 업로드
- 중복 행, 빈 값, 헤더 공백, 숫자 형식, 날짜 형식 검사
- 공백 정리, 열 이름 표준화, 숫자 정리, 중복 제거
- 선택한 열 기준 요약표 생성
- 두 파일을 같은 키 기준으로 비교
- 추가, 삭제, 변경, 동일 행 요약
- 결과 CSV 다운로드

## 기술 메모

- React + TypeScript + Vite
- 브라우저 내 파일 처리 우선
- GitHub Pages 무료 배포
- 재사용 가능한 scanner, transform, recipe 구조
