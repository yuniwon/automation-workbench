# Automation Workbench v0.1.5

무료 엑셀/CSV 도구 전체를 한 번에 보여주는 use case 허브 페이지를 추가한 버전입니다.

Tool index: https://yuniwon.github.io/automation-workbench/use-cases/

## 변경 사항

- 새 정적 페이지 추가: `public/use-cases/index.html`
- 허브 페이지에 전체 무료 도구 7개 링크 연결
- 허브 페이지에 OpenGraph, Twitter Card, `CollectionPage` JSON-LD 추가
- `sitemap.xml`에 `https://yuniwon.github.io/automation-workbench/use-cases/` 등록
- README와 공유 키트에 도구 모음 링크 추가
- 생성기 테스트에서 허브 페이지와 sitemap 동기화 검증

## 기대 효과

- 외부 게시글에서 하나의 대표 링크로 전체 도구 목록을 보여줄 수 있다.
- 검색엔진이 개별 도구 페이지 간 관계를 더 쉽게 이해할 수 있다.
- 향후 무료 도구가 늘어나도 생성기 한 곳에서 허브와 sitemap을 같이 갱신할 수 있다.

## 검증

- `pnpm test`

- IndexNow 제출 확인
