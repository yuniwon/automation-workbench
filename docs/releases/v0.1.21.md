# Automation Workbench v0.1.21

This release adds a copy-ready posting packet command for the free marketing channels.

## Changes

- Added `pnpm marketing:post --channel <channel>` for GeekNews, OKKY, Hacker News, and Product Hunt.
- Added `pnpm marketing:post --list` to show supported posting channels.
- Extracted title, body, comment, Product Hunt fields, and post-publication log blocks from the existing markdown packets.
- Documented the command in `docs/marketing/posts/README.md`.

## Verification

- `pnpm marketing:post --list`
