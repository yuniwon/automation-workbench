# Automation Workbench v0.1.4

무료 엑셀/CSV 도구의 공개 랜딩 페이지 메타데이터를 보강한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 변경 사항

- 모든 use case 페이지에 OpenGraph 메타데이터 추가
- 모든 use case 페이지에 Twitter Card 메타데이터 추가
- 모든 use case 페이지에 `SoftwareApplication` JSON-LD 구조화 데이터 추가
- 공유 이미지 `public/og-image.png`를 랜딩 페이지 메타데이터에 연결
- 생성기 테스트에서 공개 랜딩 페이지 메타데이터 존재 여부 검증

## 기대 효과

- 커뮤니티나 메신저에 링크를 공유할 때 제목, 설명, 이미지가 더 명확하게 표시된다.
- 검색엔진이 각 무료 도구 페이지의 성격을 더 구조적으로 읽을 수 있다.
- 외부 게시 전 링크 미리보기 품질을 높인다.

## 검증

- `pnpm test`
- `pnpm build`
- use case 페이지 생성기 테스트
- 공개 GitHub Pages 배포 확인
- 공개 use case 페이지 메타데이터 확인
