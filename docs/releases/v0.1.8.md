# Automation Workbench v0.1.8

맞춤 엑셀/CSV 자동화 제작 문의 페이지를 추가한 버전입니다.

Live service page: https://yuniwon.github.io/automation-workbench/services/excel-automation-inquiry.html

## 변경 사항

- `/services/excel-automation-inquiry.html` 정적 랜딩 페이지 추가
- 제작 범위를 `5만 원부터`, `15만 원부터`, `30만 원부터` 단계로 안내
- 문의 메일 CTA 본문에 파일 형식, 반복 작업, 수작업 소요시간, 필요한 결과물 질문 포함
- 서비스 페이지를 sitemap과 IndexNow 제출 대상에 포함
- README, share-kit, revenue-process 문서에 제작 문의 페이지 링크 추가

## 기대 효과

- 무료 도구 사용자가 바로 견적 문의 전에 준비할 정보를 확인할 수 있다.
- 문의 메일에 범위 산정에 필요한 정보가 더 많이 포함된다.
- 검색/공유용 랜딩 페이지와 실제 수익화 CTA가 분리되어 전환 흐름을 측정하기 쉽다.
