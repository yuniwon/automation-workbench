# Automation Workbench v0.1.9

메인 앱 문의 패널에서 제작 범위 페이지로 바로 이동할 수 있게 한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 변경 사항

- 메인 앱 하단 문의 패널에 `제작 범위 보기` 링크 추가
- `serviceInquiryUrl` 설정을 `contact.ts`로 분리
- 문의 패널 테스트에 서비스 페이지 링크 노출 검증 추가

## 기대 효과

- 무료 도구 사용자가 이메일을 보내기 전에 제작 범위와 준비 자료를 먼저 확인할 수 있다.
- 맞춤 제작 문의 페이지가 README나 게시글뿐 아니라 실제 앱 사용 흐름 안에서도 노출된다.
