# Automation Workbench v0.1.1

무료 엑셀/CSV 정리·비교 도구의 노출과 문의 전환을 보강한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 변경 사항

- 검색 유입용 use case 페이지 3개 추가
  - 엑셀 중복 제거
  - CSV/XLSX 파일 비교
  - 주문·정산 파일 자동화
- `sitemap.xml`에 use case 페이지 등록
- 개인정보 안내 문서 `PRIVACY.md` 추가
- 문의 패널에 브라우저 내 파일 처리 안내와 개인정보 안내 링크 추가
- IndexNow 키 파일과 URL 제출 스크립트 추가

## 맞춤 제작 문의

현재 쓰는 파일 구조에 맞춘 자동화가 필요하면 아래 메일로 문의할 수 있습니다.

```text
dnjsdndus@gmail.com
```

문의할 때는 파일 형식, 반복 작업, 현재 수작업 소요시간, 필요한 결과물,
희망 마감일, 샘플 파일 공유 가능 여부를 함께 적으면 빠르게 검토할 수 있습니다.
