# Automation Workbench v0.1.13

검색 방문자를 위한 엑셀 자동화 제작 서비스 페이지를 추가한 버전입니다.

Live service page: https://yuniwon.github.io/automation-workbench/services/excel-automation-service.html

## 변경 사항

- `엑셀 자동화 제작 서비스` 정적 페이지 추가
- `엑셀 자동화 외주`, `무료 진단`, `견적 기준` 중심의 서비스 설명 추가
- 새 서비스 페이지를 sitemap과 IndexNow 제출 대상에 포함
- README, share-kit, revenue-process에 서비스 진입점 추가
- 서비스 페이지 테스트에 새 랜딩 페이지 검증 추가

## 기대 효과

- 외부 게시 없이도 검색 유입을 받을 수 있는 제작 서비스 진입점이 늘어난다.
- 사용자가 무료 도구와 제작 문의 사이에서 서비스 범위를 더 명확히 확인할 수 있다.
- Gmail 문의가 들어왔을 때 `source=service-excel-automation-service`로 유입 경로를 구분할 수 있다.
