# Automation Workbench v0.1.11

서비스 페이지와 업무별 자동화 예시 페이지의 문의 전환 정보를 보강한 버전입니다.

Live service page: https://yuniwon.github.io/automation-workbench/services/excel-automation-inquiry.html

## 변경 사항

- 제작 문의 페이지에 `문의 전 체크리스트` 카드 추가
- 제작 문의 페이지에 `가격이 달라지는 기준` 카드 추가
- workflow 개별 페이지 3개에 같은 체크리스트와 가격 기준 추가
- 서비스 페이지와 workflow 개별 페이지에 `FAQPage` JSON-LD 추가
- 샘플 파일, 개인정보, 가격 산정 관련 FAQ를 정적 페이지에 표시

## 기대 효과

- 사용자가 문의 전에 필요한 정보를 더 명확히 확인할 수 있다.
- 가격 범위가 왜 달라지는지 설명되어 문의 품질이 좋아진다.
- 검색엔진이 제작 문의 페이지와 workflow 페이지의 FAQ를 구조화 데이터로 이해할 수 있다.
