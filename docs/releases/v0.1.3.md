# Automation Workbench v0.1.3

무료 엑셀/CSV 도구에 파일 병합 기능을 추가한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 변경 사항

- 새 무료 도구: 엑셀/CSV 파일 병합
  - 여러 CSV/XLSX 파일 선택
  - 원본 파일명 열 추가
  - 파일마다 다른 열은 전체 열 합집합으로 병합
  - 병합 결과 CSV 다운로드
- 병합 샘플 CSV 링크 추가
- 검색 유입용 랜딩 페이지 추가
  - https://yuniwon.github.io/automation-workbench/use-cases/excel-csv-file-merge.html
- README, 공유 키트, 외부 게시 원고를 정리·비교·병합 기준으로 갱신
- 다음 무료 도구 로드맵 갱신
