# Automation Workbench v0.1.12

업무별 자동화 예시 페이지에서 문의로 이어지는 경로를 더 정확히 남기는 버전입니다.

Live workflow page: https://yuniwon.github.io/automation-workbench/workflows/shopping-mall-order-cleanup.html

## 변경 사항

- workflow 페이지 CTA에 `workflow`와 `intent` 추적 파라미터 추가
- `무료 도구 열기`, `제작 범위 보기`, `맞춤 제작 문의` CTA를 구분해서 기록
- workflow 직접 문의 메일 본문에 업무 예시와 CTA 상세 정보 추가
- 제작 문의 페이지가 URL 파라미터를 읽어 메일 본문에 유입 경로 상세를 추가
- 서비스 페이지와 workflow 페이지 테스트에 CTA 추적 검증 추가

## 기대 효과

- Gmail 문의가 들어왔을 때 어떤 업무 예시에서 전환됐는지 확인할 수 있다.
- 사용자가 무료 도구를 먼저 열었는지, 제작 범위를 본 뒤 문의했는지 구분할 수 있다.
- 다음 무료 도구를 고를 때 실제 문의 의도를 기준으로 우선순위를 정할 수 있다.
