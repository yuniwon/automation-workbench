# Automation Workbench v0.1.6

견적서/정산서 자동 생성 샘플을 추가한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/?tool=report

Use case page: https://yuniwon.github.io/automation-workbench/use-cases/estimate-settlement-generator.html

## 변경 사항

- 새 도구 모드 추가: `견적서/정산서 자동 생성`
- 주문 CSV/XLSX에서 그룹 기준 열, 품목 열, 금액 열 선택
- 고객별/거래처별 행 수, 총 금액, 품목 요약 생성
- 정산 결과 CSV 다운로드
- 정산 결과 HTML 다운로드
- use case 랜딩 페이지와 sitemap 등록
- `tool=report` 쿼리 파라미터로 정산서 도구 직접 열기 지원
- README, 공유 키트, 무료 도구 로드맵 갱신

## 기대 효과

- 정리, 비교, 병합 다음 단계인 문서/리포트 생성까지 무료 도구 흐름에 포함된다.
- "우리 양식으로 정산서를 만들어주세요" 같은 유료 전환 문의를 받기 쉬워진다.
- 고객별 양식, 세금 계산, 로고, 문구, 예외 처리 같은 맞춤 제작 범위로 확장할 수 있다.
