# Automation Workbench v0.1.2

무료 엑셀/CSV 정리·비교 도구의 검색 유입과 GitHub 방문자 전환을 보강한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/

## 변경 사항

- 검색 유입용 use case 페이지 3개 추가
  - 엑셀 빈 값 검사
  - 엑셀 숫자 형식 정리
  - 엑셀 파일 비교
- use case 페이지와 `sitemap.xml`을 `scripts/use-case-pages.mjs`에서 생성하도록 정리
- README에 바로 써볼 수 있는 사용 사례 링크 표와 문의 메일 링크 추가
- 공유 키트에 6개 랜딩 페이지 링크 추가
- IndexNow 제출 대상이 루트 포함 7개 URL로 확장
