# Automation Workbench v0.1.7

문의 전환 추적과 외부 게시 원고를 보강한 버전입니다.

Live demo: https://yuniwon.github.io/automation-workbench/?source=geeknews&tool=report

## 변경 사항

- 문의 메일 본문에 유입 경로(`source`)와 선택 도구(`tool`)를 함께 기록
- 문의 메일 본문에 추적 파라미터가 포함된 도구 링크 추가
- `utm_source` 또는 `source` 기반 유입 추적 유지
- GeekNews, OKKY, Hacker News, Product Hunt 게시 원고를 정산서 생성 도구 포함 상태로 갱신
- 게시 후 Gmail 검색식에 `견적서`, `정산서` 키워드 추가

## 기대 효과

- 실제 Gmail 문의가 들어왔을 때 어떤 채널과 어떤 도구에서 전환됐는지 바로 확인할 수 있다.
- 외부 게시 원고가 현재 공개 기능과 일치한다.
- 정산서 생성 기능을 보고 들어온 문의를 별도로 분류할 수 있다.

## 검증

- `pnpm test`
