# Free Tool Roadmap

목표: 무료 도구를 늘려 검색/커뮤니티 유입을 만들고, 실제 업무 파일에 맞춘 유료 자동화 문의로 연결한다.

## 완료된 무료 도구

### 1. 엑셀/CSV 자동 정리 도구

핵심 가치:

- 중복 행, 빈 값, 숫자 형식, 헤더 문제를 브라우저에서 확인한다.
- 공백 정리, 숫자 정규화, 중복 제거 후 CSV로 내려받는다.

판매로 이어지는 문장:

> 반복 정리 규칙, 결과 양식, 예외 처리가 정해져 있다면 고객 파일에 맞춘 자동화 도구로 제작할 수 있습니다.

### 2. 엑셀/CSV 파일 비교 도구

핵심 가치:

- 두 CSV/XLSX 파일을 같은 키 기준으로 비교한다.
- 추가, 삭제, 변경, 동일 행을 요약하고 결과 CSV를 내려받는다.

판매로 이어지는 문장:

> 실제 정산, 주문, 재고 파일은 매칭 기준과 예외 규칙이 다르기 때문에 업무 파일에 맞춘 비교 자동화로 제작할 수 있습니다.

### 3. 엑셀/CSV 파일 병합 도구

핵심 가치:

- 여러 CSV/XLSX 파일을 세로로 병합한다.
- 원본 파일명 열을 추가해 어떤 파일에서 온 행인지 유지한다.
- 파일마다 다른 열은 전체 열 합집합으로 맞춘다.

판매로 이어지는 문장:

> 파일명에서 날짜나 매장명을 추출하고, 병합 후 정리·비교·요약까지 이어지는 반복 업무 자동화로 확장할 수 있습니다.

### 4. 견적서/정산서 자동 생성 샘플

핵심 가치:

- 샘플 주문 데이터 또는 업로드 파일에서 그룹/품목/금액 열을 선택한다.
- 고객별 또는 거래처별 정산 요약을 만든다.
- 결과를 CSV 또는 HTML 문서로 내려받는다.

판매로 이어지는 문장:

> 고객사 양식, 로고, 세금 계산 규칙, 예외 처리가 정해져 있다면 반복 문서 생성 업무를 맞춤 자동화로 제작할 수 있습니다.

## 다음 후보

### 1순위: 구글시트 검수 자동화 샘플

이유:

- 실제 고객 업무는 파일 업로드보다 구글시트 반복 작업일 수 있다.
- 다만 OAuth/권한/백엔드가 필요할 수 있어 무료 정적 도구보다 구현 비용이 크다.

MVP:

- 구글시트 예시 데이터를 CSV로 붙여넣거나 업로드
- 필수 열 누락, 중복, 숫자 형식 검사
- 검수 결과 리포트 다운로드
- 실제 구글시트 자동화 제작 문의 CTA

### 2순위: 전화번호/주소 정리 도구

이유:

- 쇼핑몰/고객명단 업무와 맞다.
- 개인정보 처리 신뢰 문구가 더 중요해진다.

### 3순위: 파일명 기반 날짜/매장명 추출 도구

이유:

- 병합 도구와 자연스럽게 이어진다.
- 매장별, 월별 파일 통합 업무에서 실제 수요가 있다.
