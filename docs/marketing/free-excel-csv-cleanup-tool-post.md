# 무료 엑셀/CSV 정리·비교 도구를 만들었습니다

반복해서 정리하거나 서로 비교하는 주문, 정산, 재고 파일이 있다면 아래 도구를 먼저 써볼 수 있습니다.

https://yuniwon.github.io/automation-workbench/

## 할 수 있는 일

- CSV 또는 XLSX 파일 업로드
- 중복 행 검사
- 빈 값 검사
- 숫자 형식 정리
- 열 이름 공백 정리
- 선택한 열 기준 요약표 생성
- 두 파일을 같은 키 기준으로 비교
- 추가, 삭제, 변경된 행 확인
- 정리된 결과 CSV 다운로드

브라우저에서 바로 실행되는 무료 도구라 별도 설치나 계정이 필요 없습니다.

## 왜 만들었나요?

작은 사업장이나 개인 업무에서는 엑셀 파일을 매번 비슷한 방식으로 손보는 일이 많습니다.
예를 들어 주문 파일에서 중복을 지우고, 빈 값을 확인하고, 숫자 형식을 맞춘 뒤,
정산이나 발송에 쓸 파일을 다시 만들 수 있습니다. 또는 주문 파일과 정산 파일을
같은 주문번호 기준으로 비교해 누락되거나 변경된 행을 찾을 수도 있습니다.

이런 작업은 한두 번이면 손으로 해도 되지만, 매주 반복되면 시간이 계속 새어나갑니다.
그래서 먼저 누구나 써볼 수 있는 작은 자동 정리 도구를 공개했습니다.

## 맞춤 제작도 가능합니다

무료 도구는 공통적인 정리 흐름을 보여주는 샘플입니다.
실제 업무에서는 파일 양식, 열 이름, 검수 규칙, 결과 양식이 모두 다르기 때문에
업무 파일에 맞춘 자동화가 필요할 수 있습니다.

예시:

- 쇼핑몰 주문 파일을 정리해서 택배 업로드 양식으로 변환
- 여러 매장 매출 파일을 합쳐 월간 요약표 생성
- 정산 파일과 주문 파일을 비교해서 누락/차액 찾기
- 구글시트에 새 데이터가 들어오면 자동 검수 후 알림 발송
