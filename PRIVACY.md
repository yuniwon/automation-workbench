# Privacy Notes

Last updated: 2026-05-14

Automation Workbench is a static GitHub Pages tool for cleaning and comparing CSV/XLSX files.

## File Handling

- Uploaded CSV/XLSX files are parsed in the browser.
- The public tool does not upload your files to an application server.
- The public tool does not require login, account creation, or email entry to run.
- Exported CSV files are generated in the browser and downloaded by your browser.

## What May Leave Your Browser

Normal web hosting and browser behavior still applies:

- Your browser requests the static app files from GitHub Pages.
- Your browser may request public sample CSV files if you click sample links.
- If you click the inquiry email button, your email client opens a prefilled message to `dnjsdndus@gmail.com`.
- Any sample file or business data you attach to that email is sent by email, not by the public web tool.

## Sensitive Data

Before sending sample files by email:

- Remove or mask names, phone numbers, addresses, account numbers, and customer identifiers.
- Replace real amounts or order IDs with sample values if the exact values are not needed.
- Share a small representative sample first, not a full production export.

## Custom Automation Work

If custom automation is requested, file handling, storage, credentials, third-party integrations, and recurring jobs must be scoped separately before work starts.

For custom work inquiries, email `dnjsdndus@gmail.com`.
